About This Project

This music library website demonstrates the use of XML, XPath, and XSLT technologies:

  • XML (music.xml): Stores structured data about songs including titles, artists, albums, and debut years.
  • XSLT (music.xsl): Transforms the XML data into an interactive HTML webpage with styling.
  • XPath: Used within the XSLT to query specific data from the XML document.

XPath Queries Demonstrated

The following XPath queries are demonstrated in the transformed page:

  1. Select all song titles: //songTitle
  2. Select the artist for the fifth song: //song[5]/artist
  3. Select the song with id="1": //song[@id=1]
